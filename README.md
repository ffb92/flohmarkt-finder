# 🗺️ Flohmarkt-Finder

Finde Flohmärkte, Trödelmärkte & Wochenmärkte in deiner Nähe — schnell, kostenlos und ohne Anmeldung.

## Tech Stack

- **Flutter** (Android + Web)
- **Supabase** (PostgreSQL + PostGIS)
- **OpenStreetMap** (flutter_map + OSRM)
- **Vercel** (Web-Deployment)

## Live

🌐 Web: [flohmarkt-finder-alpha.vercel.app](https://flohmarkt-finder-alpha.vercel.app)

## Development

```bash
flutter pub get
flutter run          # Web
flutter build apk    # Android APK
flutter build web    # Web Build
```
